# 🌍 Air Quality Analysis App

A **no-code AI app** built with **AWS PartyRock** to analyze AQI trends, pollutants, and seasonal patterns (2023–2024).

## 📊 Features
- 📈 **Monthly AQI trends** – Visualize how air quality changes month-to-month  
- 🌦 **Seasonal analysis** – Understand how seasons affect air quality  
- 🏆 **Dominant pollutant identification** – Pinpoint the main contributors to poor AQI  
- 🌡 **Correlation with temperature & humidity** – Explore environmental factors  
- 🏙 **City-wise comparison** – Compare AQI across different locations
